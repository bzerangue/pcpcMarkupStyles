Cross Browser Sticky Footer Technique with jQuery

Here is the article where this Cross Browser Sticky Footer techinique is explained

This tweaked example has an upgraded jQuery (latest source) to be compatible and work with Opera.

Also, this example, has been adjusted to address the page load and cache issues, adjusting it from document.ready to window.load like so...

$(window).load(function(){
$("#footer").stickyFooter();
});
